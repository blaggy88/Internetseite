# EDEKA Niemerszein Website

Modernisierte Website für EDEKA Niemerszein Hamburg.

## Features
- Modernes, responsives Design
- Optimiert für Mobile & Desktop
- Interaktive Einkaufsliste
- Newsletter-Anmeldung
- Touch-freundlicher Slider

## Seiten
- `index.html` - Startseite
- `angebot.html` - Aktuelle Angebote
- `markt.html` - Märkte & Standorte

## GitHub Pages
Diese Website ist für GitHub Pages optimiert. Aktivieren Sie GitHub Pages in den Repository-Einstellungen unter "Pages" und wählen Sie den main-Branch aus.
